Software Highlights

Hemocytometer Sidekick - Easier cell counting with a hemocytometer. Available in iOS/Android/Windows app stores. Source not yet public.

SimpleANOVA.jl - Analysis of Variance package for the Julia language, for those who prefer to stay away from linear models.

Q-SCAn - Package implementing Quantitative Subcellular Compartmentalization Analysis developed for use with Cell Profiler and ImageJ to quantify the distribution of a protein among cellular compartments.

Scryptonite - Built the scrypt cryptographic algorithm from the source paper and released it as a component for Universal Windows Platform apps. Available on Nuget.
